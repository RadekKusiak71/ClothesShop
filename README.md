## ClothesShop
  Clothes shop application

# Installing
1. Clone repository
  ```
  git init
  git clone {link to repository}
  ```
2.Create virtualenv
3.Install libraries
  ```
  pip install -r requirements.txt
  ```
4.To make emails work you have to configure email backend in ecommerce/settings.py (its on the bottom of a file)
